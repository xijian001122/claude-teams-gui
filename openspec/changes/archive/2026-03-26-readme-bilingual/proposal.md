## Why

当前 README.md 仅包含英文内容且信息不完整，缺乏项目背景介绍和详细使用教程。这导致新用户难以快速理解项目价值和使用方式。需要重构为中英双语版本，增加背景介绍和完整的使用教程。

## What Changes

- 重构 README.md 为中英双语结构（中文为主，英文为辅）
- 增加项目背景章节，说明 Claude Agent GUI 的设计目的和使用场景
- 增加详细安装配置教程
- 增加功能使用说明（团队管理、消息通知、主题设置等）
- 增加常见问题解答（FAQ）
- 保持与 CHANGELOG.md、docs/ 的链接完整性

  - 详细使用教程章节
